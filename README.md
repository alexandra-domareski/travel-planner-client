# Voyago Frontend ✈️

Voyago is a modern travel planning web application that helps users organize their trips in one place.

The frontend is built with **React and TypeScript**, focusing on a clean UI, intuitive trip management, and interactive planning tools such as itinerary builders, budget tracking, and travel dashboards.

---

# 🌍 Overview

The frontend provides the user interface for the Voyago platform, allowing users to:

- create and manage trips
- plan daily itineraries
- track travel expenses
- save places to visit
- organize packing lists
- store travel documents

The application is designed with a **modern dashboard layout and responsive UI** to create a smooth user experience.

# ✨ Main Features

## Trips Dashboard

Users can view all their trips in a visual dashboard.

Features include:

- create trip
- edit trip
- delete trip
- view trip overview

Each trip card displays:

- destination
- travel dates
- cover image
- status

---

## Trip Overview

Each trip has a central dashboard showing:

- trip details
- countdown to trip
- quick budget summary
- upcoming itinerary activities

---

## Itinerary Planner

Users can organize activities day by day.

Features:

- add activity
- set time and location
- add notes
- reorder activities with drag & drop

---

## Places to Visit

Users can save locations they want to visit during the trip.

Features:

- add places
- categorize locations
- view saved places list
- map view (planned)

---

## Budget Tracker

Allows users to manage travel expenses.

Features:

- add expenses
- categorize spending
- view budget summary
- visual charts of expenses

---

## Packing List

Users can create packing checklists for their trips.

Categories include:

- clothes
- electronics
- toiletries
- documents
